# E-Commerce-Sales-Analytics

**Tools:** Python (Pandas, Seaborn, Matplotlib, SciPy) · SQL (SQLite) · Power BI · Excel  
**Dataset:** 5,000 orders · 12 variables · 4 product categories · 4 regions

## Project Summary
End-to-end data analysis project covering data cleaning, exploratory analysis, 
statistical hypothesis testing, and interactive dashboard development on a 
real-world e-commerce dataset.

## Key Findings
- Electronics generates 35.3% of revenue but Beauty has the highest avg order value ($1,090)
- High discounts reduce avg revenue by 22.7% — discounting is not driving volume gains
- T-test (p=0.199) shows delivery speed does NOT significantly affect customer ratings
- Repeat customers generate 2.24× more total revenue than one-time buyers
- Top 49% of customers drive 80% of revenue
